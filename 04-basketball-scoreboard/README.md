# 04 · Basketball Scoreboard

An interactive scoreboard for two teams, built with HTML, CSS, and vanilla
JavaScript.

## Features

- Two teams: **Home** and **Guest**
- `+1`, `+2`, `+3` buttons per team
- Reset button to zero both scores
- The leading team is highlighted with a green glow
- Responsive layout (stacks nicely on small screens)

## Concepts practiced

- DOM selection and `textContent` updates
- Event delegation via `data-*` attributes
- Keeping UI in sync with a small state object
- Conditional class toggling (`classList`)

## Run it

Open `index.html` in a browser. No build step required.
