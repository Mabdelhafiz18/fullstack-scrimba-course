# Scrimba Fullstack Development Projects

A portfolio of small, self-contained projects built while completing the
**Scrimba / Coursera Fullstack Development Specialization**. Each project lives
in its own folder with its own `README.md`, source code, and (where relevant)
its own `package.json`.

The goal is breadth: every project focuses on one slice of the curriculum, from
plain HTML/CSS through to a TypeScript Next.js app.

> Open [`index.html`](./index.html) in a browser for a simple landing page that
> links to all 12 projects.

## Running the projects

- **Static projects (01–07):** open `index.html` in a browser, or serve the
  folder with a static server (e.g. `npx serve .`).
- **React projects (08, 09):** `cd` into the folder, run `npm install`, then
  `npm run dev`.
- **Express API (10):** `cd` in, `npm install`, then `npm run dev`.
- **SQL (11):** run the `.sql` files against SQLite/Postgres/MySQL.
- **Next.js (12):** `cd` in, `npm install`, then `npm run dev`.
