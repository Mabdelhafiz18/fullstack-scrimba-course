# 03 · Birthday Website

A cheerful, animated birthday greeting page built with HTML and CSS.

## Features

- Gradient hero with an animated "pop-in" title
- Floating balloon animation in the background
- Three "gift" cards that unwrap on hover (lid lifts, message reveals)
- Smooth CSS transitions throughout
- Respects `prefers-reduced-motion`

## Concepts practiced

- CSS `@keyframes` animations (float, pop-in)
- Hover-triggered transitions (`max-height` reveal trick)
- Responsive grid layout
- Accessibility: disabling motion for users who prefer it

## Run it

Open `index.html` in a browser, or run `npx serve .` inside this folder.
Change the name in `index.html` to personalize it.
