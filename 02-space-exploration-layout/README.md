# 02 · Space Exploration Landing Page

A responsive marketing landing page for a fictional space agency, built with
HTML and CSS only.

## Features

- Sticky, blurred navigation header
- Hero section with a layered radial/linear gradient background
- Gradient-clipped headline text
- Call-to-action buttons with hover lift effects
- Mission cards in a responsive auto-fit grid
- Alternating content sections and a closing CTA

## Concepts practiced

- CSS gradients (radial + linear, layered)
- `clamp()` for fluid typography and spacing
- CSS Grid with `auto-fit` / `minmax`
- Hover transitions and `backdrop-filter`
- Smooth scrolling with anchor navigation

## Run it

Open `index.html` in a browser, or run `npx serve .` inside this folder.
